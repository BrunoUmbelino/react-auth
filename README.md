This project is a demonstration of Authentication by Token in React and Node. 

### Tecnologies

Front-end
* React (Functional Components, Hooks, Custom Hooks, Context API)
* Axios
* Styled Components
* Formik / Yup
* JavaScript ES6
* HTML5
* CSS3

[Back-end](https://github.com/BrunoUmbelino/node-auth)
* Node
* Express
* MongoDB Atlas
* Mongoose
* Axios
* JsonWebToken
* BcryptJs
